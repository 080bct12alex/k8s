

- run each containers in separate pods
   - connect mongodb using host,port through env   

-  refered image : node-mongo-app:02 
   - as express connect mongodb using host i.e service name of mongodb,port through env 

###
COMMANDS: (order matter)    
    kubectl apply -f mongo-db.yml
    kubectl get all
    kubectl apply -f mongo-config.yml
    kubectl apply -f node-mongo-app.yml
    minikube service service-node-mongo-app-multi-pod

Monitor the status of  deployments, pods, and services.
     minikube dashboard
